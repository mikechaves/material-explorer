# Material Explorer Docs

This directory is the planning and technical documentation home for Material Explorer.

## Source Of Truth

- [Active Backlog](./backlog/ACTIVE_BACKLOG.md): short queue for work we are willing to start next.
- [Future Backlog](./backlog/FUTURE_BACKLOG.md): deferred, long-range, or decision-bound work.
- [Backend Readiness Plan](./backend-readiness.md): API and persistence migration notes.
- [Full Stack Audit](./full-audit-2026-02-14.md): historical audit findings and implemented progress.

Backlog files are the canonical work queues. Other docs can contain decisions, evidence, runbooks, or
technical context, but follow-up work should be promoted to the active backlog or parked in the future
backlog before it is treated as current scope.
